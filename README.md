# leetcode-dashboard
Responsive LeetCode stats tracker built with HTML, CSS, and JavaScript using LeetCode API
 LeetCode Stats Tracker

A responsive web application that allows users to search any LeetCode username and view their problem-solving statistics in an attractive dashboard UI.

#Features:
 Search LeetCode users by username
 Shows Easy, Medium, Hard progress
 Total solved problems
 Fully responsive for mobile, tablet & desktop
 Fast and lightweight UI
 
 #Built With:(HTML,CSS,JavaScript,LeetCode API)
